# Exercício 1 — Função ou procedimento?

- **Str() — Função:** recebe um número e retorna esse valor convertido em uma string.
- **QOut() — Procedimento:** executa a ação de exibir um conteúdo na tela, sem produzir um valor útil para o programa.
- **Date() — Função:** retorna a data atual do sistema.
- **Len() — Função:** retorna a quantidade de caracteres de uma string ou de elementos de um array.
- **Upper() — Função:** retorna uma nova string convertida para letras maiúsculas.
- **ClearScreen() — Procedimento:** apenas executa a ação de limpar a tela.
